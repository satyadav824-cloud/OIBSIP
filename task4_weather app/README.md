# 🌦️ Weather Dashboard 2.0

A Python-based Weather Dashboard that fetches live weather information using the OpenWeather API.

## ✨ Features

- 🌍 Search weather by city name
- 🌡️ Temperature
- 💧 Humidity
- 🌬️ Wind Speed
- 🌅 Sunrise Time
- 🌇 Sunset Time
- 📝 Weather Description
- ❌ Error Handling
- 🎨 Clean Command-Line Interface

## 🛠️ Technologies Used

- Python
- Requests Library
- OpenWeather API

## ▶️ How to Run

1. Install requests library

```bash
pip install requests
```

2. Run the project

```bash
python weather_app.py
```
