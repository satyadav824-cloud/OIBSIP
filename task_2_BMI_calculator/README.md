# 🧮 BMI Calculator

A simple Python-based BMI Calculator that calculates the Body Mass Index (BMI) using a user's height and weight.

## ✨ Features

- Calculate BMI
- Health Category Detection
- Input Validation
- User-Friendly CLI

## 🛠️ Technologies Used

- Python

## ▶️ How to Run

```bash
python bmi_calculator.py
```
